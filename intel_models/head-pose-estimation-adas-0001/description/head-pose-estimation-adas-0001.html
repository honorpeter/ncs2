Overview of OpenVINO™ toolkit Pre-trained Models

head-pose-estimation-adas-0001

Use Case and High-Level Description

Head pose estimation network based on trimmed SqueezeNet_v1.1. Angle regression layers are convolutions (width: 128) + ReLU + batch norm + fully connected with one output.

Validation Dataset

Biwi Kinect Head Pose Database

Inputs

Image in BGR format with size 60x60.

Outputs

Output layer names in Inference Engine format:

  • angle_y_fc
  • angle_p_fc
  • angle_r_fc

Output layer names in Caffe* format:

  • fc_y
  • fc_p
  • fc_r

Each output contains one float value that represents value in Tait-Bryan angles (yaw, pitсh or roll).
